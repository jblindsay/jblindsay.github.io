Rescale image value range

This tool can be used to rescale the values contained within a raster to a user-specified range.

See Also:

  • None

Scripting:

The following is an example of a Python script using this tool:

wd = pluginHost.getWorkingDirectory()
inputFile = wd + "input.dep"
outputFile = wd + "output.dep"
newMinimum = "0.0"
newMaximum = "100.0"
args = [inputFile, outputFile, newMinimum, newMaximum]
pluginHost.runPlugin("RescaleImageValueRange", args, False)

This is a Groovy script also using this tool:

def wd = pluginHost.getWorkingDirectory()
def inputFile = wd + "input.dep"
def outputFile = wd + "output.dep"
def newMinimum = "0.0"
def newMaximum = "100.0"
String[] args = [inputFile, outputFile, newMinimum, newMaximum]
pluginHost.runPlugin("RescaleImageValueRange", args, false)
