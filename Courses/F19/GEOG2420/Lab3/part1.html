Enhancing Multispectral Satellite Images

Warning, the data that are used in this lab are quite large. Do not try to run this lab on your personal laptop if it does not have sufficient memory, available hard drive space, and processing power. For example, it is unlikely that a MacBook Air would be sufficiently capable to perform the analyses carried out in this lab exercise. If you find yourself in this situation, use the workstations in the undergraduate computing lab instead.

Open QGIS and display the raster file band2.tif, band3.tif, and band4.tif. These three images comprise three of the available bands within a Landsat 8 data set collected on 2019-10-12. Using the CreateColourComposite tool (available from WhiteboxTools => Image Processing Tools in QGIS), create a true colour composite using RGB = 4 3 2. Be sure to set the Input Opacity Band to nothing and check the Perform balance contrast enhancement? checkbox. Call the output RGB432.tif, being sure to save the file in your lab working directory.

Notice that the colours within your composite image appear to be somewhat faded and subdued. Use the DirectDecorrelationStretch tool (WhiteboxTools => Image Processing Tools - Image Enhancement) to perform a saturation stretch on your true colour composite image. You may use the default parameter values; call the output file RGB432_dds.tif and be sure to save it in your working directory.

Now use the UnsharpMasking tool (WhiteboxTools => Image Processing Tools - Filters). Despite the counter-intuitive name of this operation, unsharp masking is used to enhance the sharpness of edges within an image. Again, you may use the default parameters, specifying the input image as RGB432_dds.tif and naming the output RGB432_unsharp.tif.

Use the PanchromaticSharpening tool (WhiteboxTools => Image Processing Tools - Image Enhancement) to combine the band 8 data with the RGB432_unsharp.tif colour composite image. Do not specify the red, green, or blue bands (i.e. remove any auto-populated files names that have been placed in the first three parameters in the PanchromaticSharpening dialog) and specify RGB432_unsharp.tif as the input colour composite image. You may use the default Brovey pan-sharpening method; call the output image pansharp.tif.

You will undoubtedly notice that the output of the operation produces an image that is very dark, with an unsatisfactory colour-balance. Use the BalanceContrastEnhancement tool (WhiteboxTools => Image Processing Tools - Image Enhancement) to improve the quality of the pan-sharpened image. Call the output pansharp_bce.tif.
