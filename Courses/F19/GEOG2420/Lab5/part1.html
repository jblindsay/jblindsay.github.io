The Landsat 8 satellite has two sensors onboard, including the Operational Land Imager (OLI), which collects bands 1-9, and the Thermal Infrared Sensor (TIRS), which is responsible for imaging bands 10 and 11. The values stored within the pixels of this Landsat image are in digital numbers (DNs) converted using a quantification system for the convenience of data storage. DN values have no unit nor any physical meaning other than relative value, i.e. one pixel is relatively 'brighter' than another. Therefore, these DNs need to be converted to radiance, then to Top of Atmosphere (i.e. "at sensor") temperature (TOAbt). These data could then be further converted to land surface temperature (LST), in order to perform more quantitative analysis from the thermal remote sensing data. However, doing so would require a model of the atmosphere at the time the image was taken. For our purposes, TOAbt will suffice. TOAbt and LST are the radiative temperatures that depend on the albedo (colour), chemical composition, roughness, vegetation cover, and soil moisture. For example, a polished, shiny metal surface can have a relatively low radiative temperature while that same metal surface painted black will have a much higher radiative temperature.

The metadata file associated with this image (LC08_L1TP_018030_20190630_20190706_01_T1_MTL.txt) contains all of the information that we need to convert DNs to TOAbt:

Term Value
RADIANCE_ADD_BAND 0.1
RADIANCE_MULT_BAND 0.0003342
K1_CONSTANT_BAND_10 (K1) 774.8853
K2_CONSTANT_BAND_10 (K2) 1321.0789

To perform this conversion, you will first need to convert the DN values to Top of Atmosphere Radiance (TOAr), using the following equation:

TOAr = RADIANCE_MULT_BAND * b10 + RADIANCE_ADD_BAND`

Use the Raster Calculator in QGIS (found under the Raster menu) to perform the following raster formulation:

0.0003342 * "b10@1" + 0.1

Be sure to call the output file TOAr.tif. Notice that the name of the file "b10@1" should be inserted into the equation by selecting it from the Raster Bands textbox in the Raster Calculator. It must be displayed to be available for selection.

Next, we will convert the TOAr values to top-of-atmosphere brightness temperatures (TOAbt), in Celsius, using the following equation:

TOAbt = K2 / ln(K1 / TOAr + 1) - 273.15

This calculation can be performed in the Raster Calculator as:

1321.0789/ln(774.8853/"TOAr@1"+1)-273.15

Name the output TOAbt.tif. Note that this image is the at-sensor temperature and not the true land surface temperature. Land surface temperature can be calculate using radiative transfer simulations, e.g. LOWTRAN or MODTRAN codes, but this is further than we will go in this lab.
