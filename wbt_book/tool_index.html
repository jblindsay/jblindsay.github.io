WhiteboxTools User Manual

Index of Tools

  1. AbsoluteValue: Calculates the absolute value of every cell in a raster. Found in Math and Stats Tools.

  2. AdaptiveFilter: Performs an adaptive filter on an image. Found in Image Processing Tools → Filters.

  3. AddPointCoordinatesToTable: Modifies the attribute table of a point vector by adding fields containing each point's X and Y coordinates. Found in Data Tools.

  4. Add: Performs an addition operation on two rasters or a raster and a constant value. Found in Math and Stats Tools.

  5. AggregateRaster: Aggregates a raster to a lower resolution. Found in GIS Analysis.

  6. And: Performs a logical AND operator on two Boolean raster images. Found in Math and Stats Tools.

  7. Anova: Performs an analysis of variance (ANOVA) test on a raster dataset. Found in Math and Stats Tools.

  8. ArcCos: Returns the inverse cosine (arccos) of each values in a raster. Found in Math and Stats Tools.

  9. ArcSin: Returns the inverse sine (arcsin) of each values in a raster. Found in Math and Stats Tools.

  10. ArcTan: Returns the inverse tangent (arctan) of each values in a raster. Found in Math and Stats Tools.

  11. Aspect: Calculates an aspect raster from an input DEM. Found in Geomorphometric Analysis.

  12. Atan2: Returns the 2-argument inverse tangent (atan2). Found in Math and Stats Tools.

  13. AttributeCorrelation: Performs a correlation analysis on attribute fields from a vector database. Found in Math and Stats Tools.

  14. AttributeHistogram: Creates a histogram for the field values of a vector's attribute table. Found in Math and Stats Tools.

  15. AttributeScattergram: Creates a scattergram for two field values of a vector's attribute table. Found in Math and Stats Tools.

  16. AverageFlowpathSlope: Measures the average slope gradient from each grid cell to all upslope divide cells. Found in Hydrological Analysis.

  17. AverageNormalVectorAngularDeviation: Calculates the circular variance of aspect at a scale for a DEM. Found in Geomorphometric Analysis.

  18. AverageOverlay: Calculates the average for each grid cell from a group of raster images. Found in GIS Analysis → Overlay Tools.

  19. AverageUpslopeFlowpathLength: Measures the average length of all upslope flowpaths draining each grid cell. Found in Hydrological Analysis.

  20. BalanceContrastEnhancement: Performs a balance contrast enhancement on a colour-composite image of multispectral data. Found in Image Processing Tools → Image Enhancement.

  21. Basins: Identifies drainage basins that drain to the DEM edge. Found in Hydrological Analysis.

  22. BilateralFilter: A bilateral filter is an edge-preserving smoothing filter introduced by Tomasi and Manduchi (1998). Found in Image Processing Tools → Filters.

  23. BlockMaximumGridding: Creates a raster grid based on a set of vector points and assigns grid values using a block maximum scheme. Found in GIS Analysis.

  24. BlockMinimumGridding: Creates a raster grid based on a set of vector points and assigns grid values using a block minimum scheme. Found in GIS Analysis.

  25. BoundaryShapeComplexity: Calculates the complexity of the boundaries of raster polygons. Found in GIS Analysis → Patch Shape Tools.

  26. BreachDepressions: Breaches all of the depressions in a DEM using Lindsay's (2016) algorithm. This should be preferred over depression filling in most cases. Found in Hydrological Analysis.

  27. BreachSingleCellPits: Removes single-cell pits from an input DEM by breaching. Found in Hydrological Analysis.

  28. BufferRaster: Maps a distance-based buffer around each non-background (non-zero/non-nodata) grid cell in an input image. Found in GIS Analysis → Distance Tools.

  29. Ceil: Returns the smallest (closest to negative infinity) value that is greater than or equal to the values in a raster. Found in Math and Stats Tools.

  30. CentroidVector: Identifes the centroid point of a vector polyline or polygon feature or a group of vector points. Found in GIS Analysis.

  31. Centroid: Calculates the centroid, or average location, of raster polygon objects. Found in GIS Analysis.

  32. ChangeVectorAnalysis: Performs a change vector analysis on a two-date multi-spectral dataset. Found in Image Processing Tools.

  33. CircularVarianceOfAspect: Calculates the circular variance of aspect at a scale for a DEM. Found in Geomorphometric Analysis.

  34. ClassifyOverlapPoints: Classifies or filters LAS points in regions of overlapping flight lines. Found in LiDAR Tools.

  35. CleanVector: Removes null features and lines/polygons with fewer than the required number of vertices. Found in Data Tools.

  36. ClipLidarToPolygon: Clips a LiDAR point cloud to a vector polygon or polygons. Found in LiDAR Tools.

  37. ClipRasterToPolygon: Clips a raster to a vector polygon. Found in GIS Analysis → Overlay Tools.

  38. Clip: Extract all the features, or parts of features, that overlap with the features of the clip vector. Found in GIS Analysis → Overlay Tools.

  39. Closing: A closing is a mathematical morphology operation involving an erosion (min filter) of a dilation (max filter) set. Found in Image Processing Tools.

  40. Clump: Groups cells that form discrete areas, assigning them unique identifiers. Found in GIS Analysis.

  41. CompactnessRatio: Calculates the compactness ratio (A/P), a measure of shape complexity, for vector polygons. Found in GIS Analysis → Patch Shape Tools.

  42. ConservativeSmoothingFilter: Performs a conservative-smoothing filter on an image. Found in Image Processing Tools → Filters.

  43. ConstructVectorTin: Creates a vector triangular irregular network (TIN) for a set of vector points. Found in GIS Analysis.

  44. ConvertNodataToZero: Converts nodata values in a raster to zero. Found in Data Tools.

  45. ConvertRasterFormat: Converts raster data from one format to another. Found in Data Tools.

  46. CornerDetection: Identifies corner patterns in boolean images using hit-and-miss pattern matching. Found in Image Processing Tools → Filters.

  47. CorrectVignetting: Corrects the darkening of images towards corners. Found in Image Processing Tools → Image Enhancement.

  48. Cos: Returns the cosine (cos) of each values in a raster. Found in Math and Stats Tools.

  49. Cosh: Returns the hyperbolic cosine (cosh) of each values in a raster. Found in Math and Stats Tools.

  50. CostAllocation: Identifies the source cell to which each grid cell is connected by a least-cost pathway in a cost-distance analysis. Found in GIS Analysis → Distance Tools.

  51. CostDistance: Performs cost-distance accumulation on a cost surface and a group of source cells. Found in GIS Analysis → Distance Tools.

  52. CostPathway: Performs cost-distance pathway analysis using a series of destination grid cells. Found in GIS Analysis → Distance Tools.

  53. CountIf: Counts the number of occurrences of a specified value in a cell-stack of rasters. Found in GIS Analysis → Overlay Tools.

  54. CreateColourComposite: Creates a colour-composite image from three bands of multispectral imagery. Found in Image Processing Tools.

  55. CreateHexagonalVectorGrid: Creates a hexagonal vector grid. Found in GIS Analysis.

  56. CreatePlane: Creates a raster image based on the equation for a simple plane. Found in GIS Analysis.

  57. CreateRectangularVectorGrid: Creates a rectangular vector grid. Found in GIS Analysis.

  58. CrispnessIndex: Calculates the Crispness Index, which is used to quantify how crisp (or conversely how fuzzy) a probability image is. Found in Math and Stats Tools.

  59. CrossTabulation: Performs a cross-tabulation on two categorical images. Found in Math and Stats Tools.

  60. CsvPointsToVector: Converts a CSV text file to vector points. Found in Data Tools.

  61. CumulativeDistribution: Converts a raster image to its cumulative distribution function. Found in Math and Stats Tools.

  62. D8FlowAccumulation: Calculates a D8 flow accumulation raster from an input DEM. Found in Hydrological Analysis.

  63. D8MassFlux: Performs a D8 mass flux calculation. Found in Hydrological Analysis.

  64. D8Pointer: Calculates a D8 flow pointer raster from an input DEM. Found in Hydrological Analysis.

  65. DInfFlowAccumulation: Calculates a D-infinity flow accumulation raster from an input DEM. Found in Hydrological Analysis.

  66. DInfMassFlux: Performs a D-infinity mass flux calculation. Found in Hydrological Analysis.

  67. DInfPointer: Calculates a D-infinity flow pointer (flow direction) raster from an input DEM. Found in Hydrological Analysis.

  68. Decrement: Decreases the values of each grid cell in an input raster by 1.0 (see also InPlaceSubtract). Found in Math and Stats Tools.

  69. DepthInSink: Measures the depth of sinks (depressions) in a DEM. Found in Hydrological Analysis.

  70. DevFromMeanElev: Calculates deviation from mean elevation. Found in Geomorphometric Analysis.

  71. DiffFromMeanElev: Calculates difference from mean elevation (equivalent to a high-pass filter). Found in Geomorphometric Analysis.

  72. DiffOfGaussianFilter: Performs a Difference of Gaussian (DoG) filter on an image. Found in Image Processing Tools → Filters.

  73. Difference: Outputs the features that occur in one of the two vector inputs but not both, i.e. no overlapping features. Found in GIS Analysis → Overlay Tools.

  74. DirectDecorrelationStretch: Performs a direct decorrelation stretch enhancement on a colour-composite image of multispectral data. Found in Image Processing Tools → Image Enhancement.

  75. DirectionalRelief: Calculates relief for cells in an input DEM for a specified direction. Found in Geomorphometric Analysis.

  76. Dissolve: Removes the interior, or shared, boundaries within a vector polygon coverage. Found in GIS Analysis.

  77. DistanceToOutlet: Calculates the distance of stream grid cells to the channel network outlet cell. Found in Stream Network Analysis.

  78. DiversityFilter: Assigns each cell in the output grid the number of different values in a moving window centred on each grid cell in the input raster. Found in Image Processing Tools → Filters.

  79. Divide: Performs a division operation on two rasters or a raster and a constant value. Found in Math and Stats Tools.

  80. DownslopeDistanceToStream: Measures distance to the nearest downslope stream cell. Found in Hydrological Analysis.

  81. DownslopeFlowpathLength: Calculates the downslope flowpath length from each cell to basin outlet. Found in Hydrological Analysis.

  82. DownslopeIndex: Calculates the Hjerdt et al. (2004) downslope index. Found in Geomorphometric Analysis.

  83. EdgeDensity: Calculates the density of edges, or breaks-in-slope within DEMs. Found in Geomorphometric Analysis.

  84. EdgePreservingMeanFilter: Performs a simple edge-preserving mean filter on an input image. Found in Image Processing Tools → Filters.

  85. EdgeProportion: Calculate the proportion of cells in a raster polygon that are edge cells. Found in GIS Analysis → Patch Shape Tools.

  86. ElevAbovePit: Calculate the elevation of each grid cell above the nearest downstream pit cell or grid edge cell. Found in Geomorphometric Analysis.

  87. ElevPercentile: Calculates the elevation percentile raster from a DEM. Found in Geomorphometric Analysis.

  88. ElevRelativeToMinMax: Calculates the elevation of a location relative to the minimum and maximum elevations in a DEM. Found in Geomorphometric Analysis.

  89. ElevRelativeToWatershedMinMax: Calculates the elevation of a location relative to the minimum and maximum elevations in a watershed. Found in Geomorphometric Analysis.

  90. ElevationAboveStreamEuclidean: Calculates the elevation of cells above the nearest (Euclidean distance) stream cell. Found in Hydrological Analysis.

  91. ElevationAboveStream: Calculates the elevation of cells above the nearest downslope stream cell. Found in Hydrological Analysis.

  92. EliminateCoincidentPoints: Removes any coincident, or nearly coincident, points from a vector points file. Found in GIS Analysis.

  93. ElongationRatio: Calculates the elongation ratio for vector polygons. Found in GIS Analysis → Patch Shape Tools.

  94. EmbossFilter: Performs an emboss filter on an image, similar to a hillshade operation. Found in Image Processing Tools → Filters.

  95. EqualTo: Performs a equal-to comparison operation on two rasters or a raster and a constant value. Found in Math and Stats Tools.

  96. ErasePolygonFromLidar: Erases (cuts out) a vector polygon or polygons from a LiDAR point cloud. Found in LiDAR Tools.

  97. ErasePolygonFromRaster: Erases (cuts out) a vector polygon from a raster. Found in GIS Analysis → Overlay Tools.

  98. Erase: Removes all the features, or parts of features, that overlap with the features of the erase vector polygon. Found in GIS Analysis → Overlay Tools.

  99. EuclideanAllocation: Assigns grid cells in the output raster the value of the nearest target cell in the input image, measured by the Shih and Wu (2004) Euclidean distance transform. Found in GIS Analysis → Distance Tools.

  100. EuclideanDistance: Calculates the Shih and Wu (2004) Euclidean distance transform. Found in GIS Analysis → Distance Tools.

  101. Exp2: Returns the exponential (base 2) of values in a raster. Found in Math and Stats Tools.

  102. Exp: Returns the exponential (base e) of values in a raster. Found in Math and Stats Tools.

  103. ExportTableToCsv: Exports an attribute table to a CSV text file. Found in Data Tools.

  104. ExtendVectorLines: Extends vector lines by a specified distance. Found in GIS Analysis.

  105. ExtractNodes: Converts vector lines or polygons into vertex points. Found in GIS Analysis.

  106. ExtractRasterValuesAtPoints: Extracts the values of raster(s) at vector point locations. Found in GIS Analysis.

  107. ExtractStreams: Extracts stream grid cells from a flow accumulation raster. Found in Stream Network Analysis.

  108. ExtractValleys: Identifies potential valley bottom grid cells based on local topolography alone. Found in Stream Network Analysis.

  109. FarthestChannelHead: Calculates the distance to the furthest upstream channel head for each stream cell. Found in Stream Network Analysis.

  110. FastAlmostGaussianFilter: Performs a fast approximate Gaussian filter on an image. Found in Image Processing Tools → Filters.

  111. Fd8FlowAccumulation: Calculates an FD8 flow accumulation raster from an input DEM. Found in Hydrological Analysis.

  112. Fd8Pointer: Calculates an FD8 flow pointer raster from an input DEM. Found in Hydrological Analysis.

  113. FeaturePreservingSmoothing: Reduces short-scale variation in an input DEM using a modified Sun et al. (2007) algorithm. Found in Geomorphometric Analysis.

  114. FetchAnalysis: Performs an analysis of fetch or upwind distance to an obstacle. Found in Geomorphometric Analysis.

  115. FillBurn: Burns streams into a DEM using the FillBurn (Saunders, 1999) method. Found in Hydrological Analysis.

  116. FillDepressions: Fills all of the depressions in a DEM. Depression breaching should be preferred in most cases. Found in Hydrological Analysis.

  117. FillMissingData: Fills NoData holes in a DEM. Found in Geomorphometric Analysis.

  118. FillSingleCellPits: Raises pit cells to the elevation of their lowest neighbour. Found in Hydrological Analysis.

  119. FilterLidarClasses: Removes points in a LAS file with certain specified class values. Found in LiDAR Tools.

  120. FilterLidarScanAngles: Removes points in a LAS file with scan angles greater than a threshold. Found in LiDAR Tools.

  121. FindFlightlineEdgePoints: Identifies points along a flightline's edge in a LAS file. Found in LiDAR Tools.

  122. FindLowestOrHighestPoints: Locates the lowest and/or highest valued cells in a raster. Found in GIS Analysis.

  123. FindMainStem: Finds the main stem, based on stream lengths, of each stream network. Found in Stream Network Analysis.

  124. FindNoFlowCells: Finds grid cells with no downslope neighbours. Found in Hydrological Analysis.

  125. FindParallelFlow: Finds areas of parallel flow in D8 flow direction rasters. Found in Hydrological Analysis.

  126. FindPatchOrClassEdgeCells: Finds all cells located on the edge of patch or class features. Found in GIS Analysis → Patch Shape Tools.

  127. FindRidges: Identifies potential ridge and peak grid cells. Found in Geomorphometric Analysis.

  128. FlattenLakes: Flattens lake polygons in a raster DEM. Found in Hydrological Analysis.

  129. FlightlineOverlap: Reads a LiDAR (LAS) point file and outputs a raster containing the number of overlapping flight lines in each grid cell. Found in LiDAR Tools.

  130. FlipImage: Reflects an image in the vertical or horizontal axis. Found in Image Processing Tools.

  131. FloodOrder: Assigns each DEM grid cell its order in the sequence of inundations that are encountered during a search starting from the edges, moving inward at increasing elevations. Found in Hydrological Analysis.

  132. Floor: Returns the largest (closest to positive infinity) value that is less than or equal to the values in a raster. Found in Math and Stats Tools.

  133. FlowAccumulationFullWorkflow: Resolves all of the depressions in a DEM, outputting a breached DEM, an aspect-aligned non-divergent flow pointer, and a flow accumulation raster. Found in Hydrological Analysis.

  134. FlowLengthDiff: Calculates the local maximum absolute difference in downslope flowpath length, useful in mapping drainage divides and ridges. Found in Hydrological Analysis.

  135. GammaCorrection: Performs a gamma correction on an input images. Found in Image Processing Tools → Image Enhancement.

  136. GaussianContrastStretch: Performs a Gaussian contrast stretch on input images. Found in Image Processing Tools → Image Enhancement.

  137. GaussianFilter: Performs a Gaussian filter on an image. Found in Image Processing Tools → Filters.

  138. GreaterThan: Performs a greater-than comparison operation on two rasters or a raster and a constant value. Found in Math and Stats Tools.

  139. HackStreamOrder: Assigns the Hack stream order to each tributary in a stream network. Found in Stream Network Analysis.

  140. HighPassFilter: Performs a high-pass filter on an input image. Found in Image Processing Tools → Filters.

  141. HighPassMedianFilter: Performs a high pass median filter on an input image. Found in Image Processing Tools → Filters.

  142. HighestPosition: Identifies the stack position of the maximum value within a raster stack on a cell-by-cell basis. Found in GIS Analysis → Overlay Tools.

  143. Hillshade: Calculates a hillshade raster from an input DEM. Found in Geomorphometric Analysis.

  144. Hillslopes: Identifies the individual hillslopes draining to each link in a stream network. Found in Hydrological Analysis.

  145. HistogramEqualization: Performs a histogram equalization contrast enhancment on an image. Found in Image Processing Tools → Image Enhancement.

  146. HistogramMatchingTwoImages: This tool alters the cumulative distribution function of a raster image to that of another image. Found in Image Processing Tools → Image Enhancement.

  147. HistogramMatching: Alters the statistical distribution of a raster image matching it to a specified PDF. Found in Image Processing Tools → Image Enhancement.

  148. HoleProportion: Calculates the proportion of the total area of a polygon's holes relative to the area of the polygon's hull. Found in GIS Analysis → Patch Shape Tools.

  149. HorizonAngle: Calculates horizon angle (maximum upwind slope) for each grid cell in an input DEM. Found in Geomorphometric Analysis.

  150. HortonStreamOrder: Assigns the Horton stream order to each tributary in a stream network. Found in Stream Network Analysis.

  151. HypsometricAnalysis: Calculates a hypsometric curve for one or more DEMs. Found in Geomorphometric Analysis.

  152. IdwInterpolation: Interpolates vector points into a raster surface using an inverse-distance weighted scheme. Found in GIS Analysis.

  153. IhsToRgb: Converts intensity, hue, and saturation (IHS) images into red, green, and blue (RGB) images. Found in Image Processing Tools.

  154. ImageAutocorrelation: Performs Moran's I analysis on two or more input images. Found in Math and Stats Tools.

  155. ImageCorrelation: Performs image correlation on two or more input images. Found in Math and Stats Tools.

  156. ImageRegression: Performs image regression analysis on two input images. Found in Math and Stats Tools.

  157. ImageStackProfile: Plots an image stack profile (i.e. signature) for a set of points and multispectral images. Found in Image Processing Tools.

  158. ImpoundmentSizeIndex: Calculates the impoundment size resulting from damming a DEM. Found in Hydrological Analysis.

  159. InPlaceAdd: Performs an in-place addition operation (input1 += input2). Found in Math and Stats Tools.

  160. InPlaceDivide: Performs an in-place division operation (input1 /= input2). Found in Math and Stats Tools.

  161. InPlaceMultiply: Performs an in-place multiplication operation (input1 *= input2). Found in Math and Stats Tools.

  162. InPlaceSubtract: Performs an in-place subtraction operation (input1 -= input2). Found in Math and Stats Tools.

  163. Increment: Increases the values of each grid cell in an input raster by 1.0. (see also InPlaceAdd) Found in Math and Stats Tools.

  164. IntegerDivision: Performs an integer division operation on two rasters or a raster and a constant value. Found in Math and Stats Tools.

  165. IntegralImage: Transforms an input image (summed area table) into its integral image equivalent. Found in Image Processing Tools.

  166. Intersect: Identifies the parts of features in common between two input vector layers. Found in GIS Analysis → Overlay Tools.

  167. IsNoData: Identifies NoData valued pixels in an image. Found in Math and Stats Tools.

  168. Isobasins: Divides a landscape into nearly equal sized drainage basins (i.e. watersheds). Found in Hydrological Analysis.

  169. JensonSnapPourPoints: Moves outlet points used to specify points of interest in a watershedding operation to the nearest stream cell. Found in Hydrological Analysis.

  170. JoinTables: Merge a vector's attribute table with another table based on a common field. Found in Data Tools.

  171. KMeansClustering: Performs a k-means clustering operation on a multi-spectral dataset. Found in Image Processing Tools.

  172. KNearestMeanFilter: A k-nearest mean filter is a type of edge-preserving smoothing filter. Found in Image Processing Tools → Filters.

  173. KappaIndex: Performs a kappa index of agreement (KIA) analysis on two categorical raster files. Found in Math and Stats Tools.

  174. KsTestForNormality: Evaluates whether the values in a raster are normally distributed. Found in Math and Stats Tools.

  175. LaplacianFilter: Performs a Laplacian filter on an image. Found in Image Processing Tools → Filters.

  176. LaplacianOfGaussianFilter: Performs a Laplacian-of-Gaussian (LoG) filter on an image. Found in Image Processing Tools → Filters.

  179. LasToShapefile: Converts one or more LAS files into a vector Shapefile of POINT ShapeType. Found in LiDAR Tools.

  188. LidarConstructVectorTin: Creates a vector triangular irregular network (TIN) fitted to LiDAR points. Found in LiDAR Tools.

  190. LidarGroundPointFilter: Identifies ground points within LiDAR dataset using a slope-based method. Found in LiDAR Tools.

  191. LidarHexBinning: Hex-bins a set of LiDAR points. Found in LiDAR Tools.

  192. LidarHillshade: Calculates a hillshade value for points within a LAS file and stores these data in the RGB field. Found in LiDAR Tools.

  193. LidarHistogram: Creates a histogram of LiDAR data. Found in LiDAR Tools.

  194. LidarIdwInterpolation: Interpolates LAS files using an inverse-distance weighted (IDW) scheme. When the input/output parameters are not specified, the tool interpolates all LAS files contained within the working directory. Found in LiDAR Tools.

  195. LidarInfo: Prints information about a LiDAR (LAS) dataset, including header, point return frequency, and classification data and information about the variable length records (VLRs) and geokeys. Found in LiDAR Tools.

  196. LidarJoin: Joins multiple LiDAR (LAS) files into a single LAS file. Found in LiDAR Tools.

  197. LidarKappaIndex: Performs a kappa index of agreement (KIA) analysis on the classifications of two LAS files. Found in LiDAR Tools.

  198. LidarNearestNeighbourGridding: Grids LAS files using nearest-neighbour scheme. When the input/output parameters are not specified, the tool grids all LAS files contained within the working directory. Found in LiDAR Tools.

  199. LidarPointDensity: Calculates the spatial pattern of point density for a LiDAR data set. When the input/output parameters are not specified, the tool grids all LAS files contained within the working directory. Found in LiDAR Tools.

  200. LidarPointStats: Creates several rasters summarizing the distribution of LAS point data. When the input/output parameters are not specified, the tool works on all LAS files contained within the working directory. Found in LiDAR Tools.

  201. LidarRansacPlanes: Removes outliers (high and low points) in a LiDAR point cloud. Found in LiDAR Tools.

  202. LidarRemoveDuplicates: Removes duplicate points from a LiDAR data set. Found in LiDAR Tools.

  203. LidarRemoveOutliers: Removes outliers (high and low points) in a LiDAR point cloud. Found in LiDAR Tools.

  204. LidarSegmentationBasedFilter: Identifies ground points within LiDAR point clouds using a segmentation based approach. Found in LiDAR Tools.

  205. LidarSegmentation: Segments a LiDAR point cloud based on normal vectors. Found in LiDAR Tools.

  206. LidarThinHighDensity: Thins points from high density areas within a LiDAR point cloud. Found in LiDAR Tools.

  207. LidarThin: Thins a LiDAR point cloud, reducing point density. Found in LiDAR Tools.

  208. LidarTileFootprint: Creates a vector polygon of the convex hull of a LiDAR point cloud. When the input/output parameters are not specified, the tool works with all LAS files contained within the working directory. Found in LiDAR Tools.

  209. LidarTile: Tiles a LiDAR LAS file into multiple LAS files. Found in LiDAR Tools.

  210. LidarTinGridding: Creates a raster grid based on a Delaunay triangular irregular network (TIN) fitted to LiDAR points. Found in LiDAR Tools.

  211. LidarTophatTransform: Performs a white top-hat transform on a Lidar dataset; as an estimate of height above ground, this is useful for modelling the vegetation canopy. Found in LiDAR Tools.

  215. LinearityIndex: Calculates the linearity index for vector polygons. Found in GIS Analysis → Patch Shape Tools.

  216. LinesToPolygons: Converts vector polylines to polygons. Found in Data Tools.

  217. ListUniqueValues: Lists the unique values contained in a field witin a vector's attribute table. Found in Math and Stats Tools.

  218. Ln: Returns the natural logarithm of values in a raster. Found in Math and Stats Tools.

  219. Log10: Returns the base-10 logarithm of values in a raster. Found in Math and Stats Tools.
