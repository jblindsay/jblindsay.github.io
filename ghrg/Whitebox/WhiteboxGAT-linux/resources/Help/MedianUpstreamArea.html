Median upstream area

This tool computes for each stream cell the median value of the upslope areas of all cells that are located upstream (McGlynn et al., 2003).

The user must specify:
The name of the input digital elevation model (DEM).
The name of the Streams input raster. The Streams raster should contain upslope area values for all creek cells and 0 for all non-creek cells. The Streams Raster generated by the MDInf flow accumulation tool could be used.
The name of the Median Upstream Area output raster.

Grid cells possessing the NoData value in the input DEM are assigned the NoData value in the output raster. The output raster is of the float data type and continuous data scale.

References:

McGlynn, B.L., J.McDonnell, M.Stewart and J.Seibert, 2003, On the relationships between catchment scale and streamwater mean residence time, Hydrological Processes, 17: 175-181
