TopologicalBreachBurn

This tool can be used to burn a mapped vector streams layer into a raster DEM, in order to enforce flow within the DEM to coincide with known hydrology. The tool performs a generalization of the vector hydrographic data prior to rasterization to reduce the occurrence of topological errors. For a detailed description of the algorithm, see Lindsay (2016)


References:
