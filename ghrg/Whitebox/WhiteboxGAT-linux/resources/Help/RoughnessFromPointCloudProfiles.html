Roughness From Point Cloud Profiles (RFPCP)

This tool can be used to estimate surface roughness parameters (e.g. root-mean-square-height) from profiles fitted to a LiDAR point cloud. It is used to estimate field roughness parameters input in radar backscatter models. The RFPCP tool is still in actively developmental and experimental stage and is yet not intended for widespread application. Please provide feedback to the tool authors.
