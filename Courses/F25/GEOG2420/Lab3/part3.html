Part 3: Exploring Spectral Signatures

Mono Lake is the largest waterbody within the Landsat scene.

3.1. Find Mono Lake within the pansharp_bce.tif image and describe the appearance of this waterbody relative to other lakes within the scene. (2 marks)

Overlay the Lakes_points.shp vector onto the pan-sharpened image. Notice that Mono Lake corresponds with point 1 in the vector and that two other, more typical lakes within the site are also identified. Now use the Image Stack Profile tool (Whitebox Workflows => Image processing => Image stack profile) to generate a type of spectral signature for the three lakes within the points vector file. Specify bands 1-7 as the input files for the analysis and call the output file Lakes_signatures.html, being sure to save it in your Lab 3 working directory. The tool produces an HTML report, including a graph displaying the digital number (i.e. relative brightness) of the corresponding three pixels in each of the specified bands. In truth, for this 'image stack' graph to be a true spectral signature, the bands would need to be transformed into reflectance values instead of simple digital numbers (you may do this in GEOG3420). Also, the x-axis of our graph merely lists the band number rather than the actual wavelength, and this fact would likely change the shape of the three line signatures in the plot. However, this image stack is a useful means of interrogating the relative brightness of different sites and land-covers within a multispectral data set.
