Export SAGA grid

This tool can be used to export Whitebox GAT raster files to SAGA GIS raster files (*.sgrd and *.sdat). This SAGA data file format (*.sdat) is a type of binary file that is not human-readable. This file must be accompanied by the human-readable ASCII text header file (*.sgrd). The user must specify the name of one or more Whitebox raster files to be exported. The tool will create SAGA grids for each input file. Output file names are the same of the input files with different file extensions.

See Also:

  • Import SAGA Grid
  • Whitebox file formats

Scripting:

The following is an example of a Python script using this tool:

wd = pluginHost.getWorkingDirectory()
# You may have multiple input files but they must
# be separated by semicolons in the string.
inputFiles = wd + "input1.dep" + ";" + wd + "input2.dep" + ";" + wd + "input3.dep"
args = [inputFiles]
pluginHost.runPlugin("ExportSagaGrid", args, False)
