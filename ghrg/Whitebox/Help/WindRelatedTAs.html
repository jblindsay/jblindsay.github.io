Wind-related terrain attributes

Whitebox GAT includes several tools for using digital elevation models (DEMs) to examine the relative exposure/sheltering from wind within landscapes. These tools include:

  • Directional relief
  • Exposure towards a wind flux
  • Fetch analysis
  • Horizon angle
  • Relative aspect
  • Streamline terrain
