Introduction

The purpose of this lab exercise is to familiarize students with the Whitebox Workflows for Python (WbW) software as well as multispectral imagery data sources used in Earth Observation applications.

Readings and Resources

The following materials, combined with your textbook, can be used as background materials and to help in answering the assignment questions.

  • NASA Landsat Science

  • European Space Agency 2015. Sentinel-2 User Handbook

  • Lindsay, J.B. 2019. The Whitebox Workflows for Python User Manual

Before you begin

IMPORTANT INFORMATION: This series of lab assignments will require manipulation, analysis, and visualization of several remotely sensed images. We will be separating the image analysis and data visualization components. For manipulating and analyizing remotely sensed data we will use the Whitebox Workflows for Python advanced geospatial analysis platform (WbW). For data visualization, these labs do not assume any particular remote sensing software. You are free to use whatever software you are familiar with for visualizing the imagery data in the labs, including ArcGIS, QGIS, or any other spatial analysis software capable of displaying georeferenced imagery data.

The computers in the Geography Undergraduate Computing Lab, have ArcGIS and QGIS installed. If you are planning to carry out these assignments on your own computer, then it is important that you have WbW installed along with one of the data visualization packages described above. You will be provided a floating-license user ID for WbW in lab. You are advised to review the WbW User Manual before starting this lab.

In addition to the software, you will need to download the data associated with this lab exercise (instructions in the lab). The data for all of the laboratory exercises in this class are large and will require substantial data storage. You may wish to purchase a USB flash drive to dedicate to this course and to serve as the data backup. It is important that you backup all of the data for the lab assignment.

What you need to hand in

You will hand in a printed report summarizing the answer to each of the questions in the following exercise along with the necessary colour images. Notice that you will need to have paid your lab fee to have printing privileges in the Hutt building computer labs.
