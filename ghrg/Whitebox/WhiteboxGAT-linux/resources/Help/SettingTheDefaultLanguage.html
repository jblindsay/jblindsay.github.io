Setting the default language

The default language setting can be used to change the language that much of the Whitebox core user-interface label text and messages are presented in. Setting the default language will affect the menus, toolbox names, pop-up messages and many other textual elements of the program. In some cases, it will also impact the format that numerical data are presented. Generally, the default language is set by the user upon installation, i.e. the first time that the program is launched on a new machine or under a new user. After this point, the default language can always be reset by selecting View → Options and Settings and then setting the Language option. The program will have to be re-launched before the new language setting takes affect.
