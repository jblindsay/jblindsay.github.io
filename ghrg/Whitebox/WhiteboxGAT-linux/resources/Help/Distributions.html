Distributions

This tool can be used to calculate the distribution of the grid values in the input grid. The user can choose between the following distribution options:
Complete
N classes with equal class width
N classes with equal class size

The user must specify:
The name of the input grid file.
The distribution type.
The number of classes (only used in case the distribution type is not 'Complete').
The name of the output text file.
