Export GeoTIFF

This tool can be used to export a Whitebox GAT raster file to a GeoTIFF file. The user must specify the name of one or more Whitebox GAT raster files to be exported. The tool will create GeoTIFF files, with .tif extensions, for each input file. Output file names are the same of the input files.

See Also:

  • Import GeoTIFF

Scripting:

The following is an example of a Python script using this tool:

wd = pluginHost.getWorkingDirectory()
# You may have multiple input files but they must
# be separated by semicolons in the string.
inputFiles = wd + "input1.dep" + ";" + wd + "input2.dep" + ";" + wd + "input3.dep"
args = [inputFiles]
pluginHost.runPlugin("ExportGeoTIFF", args, False)

This is a Groovy script also using this tool:
